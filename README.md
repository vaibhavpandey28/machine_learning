# Machine Learning — Learning Folder

This workspace is my personal learning folder for machine learning experiments, notes, and example notebooks. I'll add datasets, notebooks, and small projects here as I learn new techniques.

## Purpose
- Keep experiments and notebooks organized while learning ML concepts.
- Collect datasets, small scripts, and example workflows.

## Repository structure
- `environment.yml`: Conda environment for reproducible experiments.
- `insurance_dataset/`: Notebook and CSV for insurance dataset experiments.
- `numpy/`: Small demos and notebooks about NumPy.
- `pandas/`: Notebooks and CSVs for Pandas practice.

## Usage & Conventions
- Notebooks: Keep runnable examples in `.ipynb` files and include a short description at the top.
- Data: Store raw CSVs under a dataset folder (do not commit large generated files).
- Environments: Use `environment.yml` to create a conda environment: `conda env create -f environment.yml`.
- Commits: Write focused commits and keep notebooks small and focused on one idea.
